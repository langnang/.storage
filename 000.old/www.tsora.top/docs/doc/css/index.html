Selectors

Basic Selectors

  • .class
    Selects all elements with class=
  • [attribute]
    用于选取带有指定属性的元素
  • [attribute*=value]
    匹配属性值包含指定值的每个元素
  • [attribute*=value]
    选取每个带有指定属性且值包含指定字符串的元素
  • [attribute=value]
    匹配属性值包含指定值的每个元素
  • [attribute|=value]
    选取每个带有指定属性的元素,该元素的值等于指定字符串(比如 "en")或以该字符串后跟连接符作为开头的字符串(比如 "en-us")
  • [attribute~=value]
    用于选取属性值中包含指定词汇的元素
  • [attribute$=value]
    匹配属性值以指定值结尾的每个元素
  • *
    Selects all elements
  • #id
    为带有指定 id 的元素设置样式
  • element
    Selects all <p> elements
  • element element
    用于选择元素内部的元素
  • element,element
    用于选取第一个指定的元素之后(不是内部)紧跟的元素
  • element+element
    用于选择(不是内部)指定的第一个元素之后紧跟的元素
  • element>element
    用于选择特定父元素
  • element1~element2
    匹配出现在 element1 后面的 element2

Pseudo-Selectors

  • ::selection
    匹配被用户选取的选取是部分
  • :active
    用于选择活动链接
  • :after
    在被选元素的内容后面插入内容
  • :before
    在被选元素的内容前面插入内容
  • :checked
    匹配每个已被选中的 input 元素(只用于单选按钮和复选框)
  • :disabled
    规定应该禁用 input 元素
  • :empty
    匹配没有子元素（包括文本节点）的每个元素
  • :enabled
    匹配每个启用的的元素(主要用于表单元素)
  • :first-child
    用于选取属于其父元素的首个子元素的指定选择器
  • :first-letter
    用于选取指定选择器的首字母
  • :first-line
    用于选取指定选择器的首行
  • :first-of-type
    匹配元素其父级是特定类型的第一个子元素
  • :focus
    用于选取获得焦点的元素
  • :hover
    用于选择鼠标指针浮动在上面的元素
  • :lang(language)
    用于选取带有以指定值开头的 lang 属性的元素
  • :last-child
    匹配属于其父元素的最后一个子元素的每个元素
  • :last-of-type
    匹配属于其父元素的特定类型的最后一个子元素的每个元素
  • :link
    用于选取未被访问的链接
  • :not
    匹配非指定元素/选择器的每个元素
  • :nth-child(n)
    匹配属于其父元素的第 N 个子元素,不论元素的类型
  • :nth-last-child(n)
    匹配属于其元素的第 N 个子元素的每个元素,不论元素的类型,从最后一个子元素开始计数
  • :nth-of-type(n)
    匹配属于父元素的特定类型的第 N 个子元素的每个元素
  • :only-child
    匹配属于其父元素的唯一子元素的每个元素
  • :only-of-type
    匹配属于其父元素的特定类型的唯一子元素的每个元素
  • :root
    匹配文档根元素,在 HTML 中,根元素始终是 html 元素
  • :target
    可用于选取当前活动的目标元素
  • :visited
    用于选取已被访问的链接

Font

    规定元素的字体系列

Positioning

  • bottom
    规定元素的底部边缘。该属性定义了定位元素下外边距边界与其包含块下边界之间的偏移
  • clear
    规定元素的哪一侧不允许其他浮动元素
  • clip
    剪裁绝对定位元素
  • cursor
    规定要显示的光标的类型(形状)
  • display
    规定元素应该生成的框的类型
  • float
    定义元素在哪个方向浮动。以往这个属性总应用于图像,使文本围绕在图像周围,不过在 CSS 中,任何元素都可以浮动。浮动元素会生成一个块级框,而不论它本身是何种元素
  • left
    规定元素的左边缘。该属性定义了定位元素左外边距边界与其包含块左边界之间的偏移
  • overflow
    规定当内容溢出元素框时发生的事情
  • position
    规定元素的定位类型
  • right
    规定元素的右边缘。该属性定义了定位元素右外边距边界与其包含块右边界之间的偏移
  • top
    规定元素的顶部边缘。该属性定义了一个定位元素的上外边距边界与其包含块上边界之间的偏移
  • visibility
    规定元素是否可见
  • z-index
    设置元素的堆叠顺序。拥有更高堆叠顺序的元素总是会处于堆叠顺序较低的元素的前面

  • 指定包含文本的相应元素中的一个音标发音
  • rest
  • 一个缩写属性设置rest-before和rest-after属性
  • rest-after
  • 一个元素的内容讲完之后，指定要休息一下或遵守韵律边界
  • rest-before
  • 一个元素的内容讲完之前，指定要休息一下或遵守韵律边界
  • voice-balance
  • 指定了左，右声道之间的平衡
  • voice-duration
  • 指定应采取呈现所选元素的内容的长度
  • voice-pitch
  • 指定平均说话的声音的音调（频率）
  • voice-pitch-range
  • 指定平均间距的变化
  • voice-rate
  • 控制语速
  • voice-stress
  • 指示着重力度
  • voice-volume
  • 语音合成是指波形输出幅度

User-interface

  • appearance

  • box-sizing

  • icon

  • nav-down

  • nav-index
    规定元素的连续导航次序 ("tabbing order")
  • nav-left
    规定当使用 nav-left 导航键时,向何处进行导航
  • nav-right
    规定当使用 nav-right 导航键时,向何处进行导航
  • nav-up

  • resize
